On/Off by Mary Henry (1968)

Original: 60" by 60" Acrylic on Canvas
Recreation: 800px by 800px Coded in CSS


Description:

Displayed above is a recreation of an abstract geometric painting painting by Mary Henry, On/Off. While the original was created using acrylic paints on a large canvas, this recreation was created entirely with HTML5 and CSS3.

What you are looking at, in terms of the shapes created via code, is a large light-grey square with a series of circles on top of it. The circles are created by taking a square and rounding the corners by 50%. Each circle has been "clipped" by an invisible circle that contains them all.

In order to create the most accurate recreation of Henry's original work, colors were determined by an online color picker tool and measurements of the original painting were taken in Adobe Illustrator.
